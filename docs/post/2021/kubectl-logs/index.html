kubectl logsに任意のログを表示する

September 8, 2021

kubectl logsはPID1の標準出力を表示するため、直接書き込んでしまえばなんでも表示できる。

1
2
3

$ kubectl exec -it pod-xxx bash
# echo 'show as stdin' > /proc/1/fd/1
# echo 'show as stderr' > /proc/1/fd/2

1
2
3
4

$ kubectl logs pod-xxx

show as stdin
show as stderr
