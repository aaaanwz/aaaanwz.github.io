公式ドキュメントの Sharing Git credentials with your containerに色々と記載があるが、非常に簡単なソリューションがあったためメモ

Mac

1
2
3
4
5

$ sudo vi ~/.ssh/config

Host github.com
    AddKeysToAgent yes
    UseKeychain yes

Windows

1
2
3

> Set-Service ssh-agent -StartupType Automatic
> Start-Service ssh-agent
> ssh-add $HOME/.ssh/id_rsa
